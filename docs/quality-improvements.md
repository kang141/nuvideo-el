# 视频质量和闪动问题优化

## 🎯 已修复的问题

### 1. 导出视频模糊
### 2. 放大缩小时轻微闪动

## 📊 优化内容

### 1. 渲染配置优化 (`src/core/render-config.ts`)

#### 修改前：
```typescript
PREVIEW_CONFIG: {
  dpr: 0.8,  // 预览降低分辨率
  imageSmoothingQuality: 'medium',
}

EXPORT_CONFIG: {
  dpr: 1.0,
  imageSmoothingQuality: 'medium',
}
```

#### 修改后：
```typescript
PREVIEW_CONFIG: {
  dpr: 1.0,  // ✅ 提升到 1.0，与导出一致，消除闪动
  imageSmoothingQuality: 'high',  // ✅ 提升到 high，减少模糊
}

EXPORT_CONFIG: {
  dpr: 1.0,
  imageSmoothingQuality: 'high',  // ✅ 提升到 high，最佳质量
}
```

**效果：**
- ✅ 预览和导出 DPR 一致，消除切换时的闪动
- ✅ 图像平滑质量提升，画面更清晰
- ✅ 预览质量提升，所见即所得

**效果：**
- ✅ 最高质量：120 Mbps，接近无损，适合专业用途
- ✅ 清晰质量：25 Mbps，高清晰度，适合日常使用
- ✅ 流畅质量：12 Mbps，平衡质量和文件大小

### 3. FFmpeg CRF 优化 (`src/hooks/editor/useFFmpegExport.ts`)

#### 修改前：
```typescript
case 'original': return 18;  // CRF 18
case 'fhd': return 23;       // CRF 23
case 'hd': return 26;        // CRF 26
```

#### 修改后：
```typescript
case 'original': return 15;  // ✅ CRF 15 (从 18 降低)，视觉无损
case 'fhd': return 20;       // ✅ CRF 20 (从 23 降低)，高质量
case 'hd': return 23;        // ✅ CRF 23 (从 26 降低)，平衡质量
```

**CRF 说明：**
- CRF 范围：0-51（越低越清晰，文件越大）
- CRF 15：视觉无损，专业级质量
- CRF 20：高质量，适合大多数场景
- CRF 23：良好质量，平衡文件大小

**效果：**
- ✅ 显著提升导出视频的清晰度
- ✅ 减少压缩伪影和模糊
- ✅ 保留更多细节

## 🔧 使用建议

### 选择合适的质量预设

1. **最高质量（120 Mbps / CRF 15）**
   - 适合：专业制作、需要后期编辑、存档保存
   - 文件大小：较大
   - 清晰度：接近无损

2. **清晰质量（25 Mbps / CRF 20）** ⭐ 推荐
   - 适合：日常使用、分享、演示
   - 文件大小：适中
   - 清晰度：高清晰度，肉眼难以察觉压缩

3. **流畅质量（12 Mbps / CRF 23）**
   - 适合：网络分享、快速预览、存储空间有限
   - 文件大小：较小
   - 清晰度：良好，适合大多数场景

## 🐛 闪动问题解决方案

### 问题原因
1. 预览 DPR (0.8) 和导出 DPR (1.0) 不一致
2. 切换时画布尺寸变化导致重新渲染
3. 不同的图像平滑质量设置

### 解决方案
1. ✅ 统一预览和导出的 DPR 为 1.0
2. ✅ 统一图像平滑质量为 'high'
3. ✅ 确保预览和导出使用相同的渲染配置

### 验证方法
1. 刷新页面（Ctrl + Shift + R）
2. 在编辑器中放大缩小视频
3. 观察是否还有闪动
4. 导出视频并检查清晰度

## 🎨 额外优化建议

如果你的电脑性能较弱，可以考虑：

1. **降低预览 DPR**（会恢复闪动问题）
   ```typescript
   PREVIEW_CONFIG: {
     dpr: 0.8,  // 降低预览分辨率
   }
   ```

2. **使用较低的质量预设**
   - 选择"流畅"而不是"最高"

3. **关闭实时预览**
   - 在编辑时暂停预览
   - 只在需要时播放
